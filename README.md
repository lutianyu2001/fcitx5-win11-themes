# Fcitx5 Win11 风格主题 | Win11 Style Themes for Fcitx5

weasel 和 fcitx5-macos 都有 Win11 风格的主题 (见下), 但是 fcitx5 Linux 版居然没人做. 所以我做了一个. 如果你喜欢的话, 请帮我点一个 ⭐ Star!

Both weasel and fcitx5-macos offer Win11 style themes (see below), yet no such theme exists for fcitx5 on Linux. So I made one. If you like it, please give it a ⭐ Star!

* weasel: <https://github.com/rime/weasel/wiki/%E7%A4%BA%E4%BE%8B#windows-11-%E9%A3%8E%E6%A0%BC>
* fcitx5-macos: <https://github.com/fcitx-contrib/fcitx5-theme-collection/blob/master/theme/win11.conf>
